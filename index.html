Description

rdfevents2rss.php is a PHP script to convert a set of events (specified using the Event Ontology) provided by a sparql endpoint into a corresponding RSS Atom feed (see RFC 4287).

The current version must be launched as a script passed to the PHP standalone interpreter and it parses the events in the opendatahacklab ontology. The sparqllib library is used to perform the sparql query.

The feed is then generated by the class in AtomFeedGenerator.php which can be profitably reused (feel free to reuse it!) to generate Atom feeds.

See also (and subscribe if interested) the generated feed and the Techical Report (italian).
